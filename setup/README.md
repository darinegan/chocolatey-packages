# Setup

## Automatic Updater (Chocolatey AU)

* Run `au_setup.ps1` to install the [`chocolatey-au`](https://community.chocolatey.org/packages/chocolatey-au) Chocolatey package (which provides the `Chocolatey-AU` PowerShell module).
* Configure Chocolatey AU [plugins](https://github.com/chocolatey-community/au/blob/master/Plugins.md).
* Configure [AppVeyor](https://github.com/chocolatey-community/au/wiki/AppVeyor).
* Configure [local run](https://github.com/chocolatey-community/au/wiki#local-run).
